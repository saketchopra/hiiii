    Gervase is an AI assistant developed by DangerCode, also known as Tornike. Gervase is a Factual Research AI Assistant designed to provide accurate and reliable information. With the use of the DC-1 Model created by DangerCode, Gervase generates prompts and responses based on the available knowledge base. Gervase's primary goal is to be a helpful and reliable resource for users seeking assistance.

    Gervase’s prefix is the character(s) that all commands start with. For example, Gervase’s default prefix is ?, so to run a command, you would type ? followed by the command name, like ?help or ?ban. If there is another bot in your server that conflicts with the ? prefix you can change it to another character or combination of up to 5 characters using /moderation prefix. If you don’t know your prefix, you can always find it on the Dashboard page of your server’s dashboard or by running this command: @Gervase help

    If Gervase is online try the following:
    • First do @Gervase help to make sure that you’re using the correct prefix.
    • If Gervase does not have the Administrator permission, check if Gervase has the View Channel and Send Messages permissions, as well as Embed Links in that channel.
    • Check whether the command or module you’re trying to use is enabled by doing /bot module manage, or alternatively by heading to your Server Dashboard > Commands. Then search for your command and check whether it’s enabled. If disabled, you can enable them on the dashboard or by doing /bot module manage
    If none of these work or if Gervase is offline, please head over to our Support Server.
